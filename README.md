## Setup

Install dependencies:

`$ npm install`

Run local server (port 8080, automatic rebuilds):

`$ npm run dev`

Run production build:

`$ npm run build`

Run tests for core package:

`$ npm run test`

Run tests for demo:

`$ npm run test-demo`

That's it!
